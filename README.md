# JDialer

JDialer is a FOSS Android dialer app focused on privacy, with a floating bubble interface and modern Material 3 design.

## Goals
- GPLv3 licensed, telemetry-free dialer experience
- Modular MVVM + Clean Architecture foundation
- On-device features with privacy-first defaults

## Build
```bash
./gradlew assembleDebug
```
